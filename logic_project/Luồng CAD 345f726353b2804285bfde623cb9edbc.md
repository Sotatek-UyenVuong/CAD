# III. LUỒNG CHATBOT (Q&A)

## Flow chính

```
User Query
   ↓
Agent_Folder
   ↓
Agent_File
   ↓
Agent_Page
   ↓
Tool (optional)
```

---

## 1. Agent_Folder

```
Input:
- folder summary
- file summaries

IF query tổng quan:
    → trả lời

ELSE:
    → chọn file liên quan → Agent_File
```

---

## 2. Agent_File

```
IF summary đủ:
    → trả lời

ELSE:
    → chuyển Agent_Page
```

---

## 3. Agent_Page (core)

### Step 1: Lấy pages

```
→ filter nhẹ theo:
    - file đã chọn
    - keyword đơn giản

→ lấy nhiều page (<= 20–30)
```

---

### Step 2: Gemini reasoning

```
Input:
- query
- list context_md

Gemini:
- tự chọn page liên quan
- trả lời
- trả page_used
```

---

### Step 3: Tool decision

```
IF:
    "bao nhiêu", "số lượng"
→ tool_count

IF:
    "diện tích"
→ tool_area

IF:
    cần ảnh
→ trả image_url
```

# IV. GLOBAL SEARCH

## Flow riêng

```
User Query (image hoặc text)
   ↓
Embedding
   ↓
Qdrant Search (toàn bộ DB)
   ↓
Top-N pages
   ↓
Fetch Mongo context
   ↓
Agent_Page (re-rank + explain)
```

---

## 1. Query Processing

```
IF image:
    → image ->text->text_embedding

IF text:
    → text_embedding
```

---

## 2. Search Qdrant

```
search toàn bộ:
- mọi folder
- mọi file

→ top 20 pages
```

---

## 3. Fetch context

```
page_id → Mongo → context_md
```

---

## 4. Agent_Page (reuse)

```
Gemini:
- chọn page đúng nhất
- giải thích vì sao match
```

---

## 5. Output

```json
{
  "file_name": "...",
  "page_number": 5,
  "image_url": "...",
  "score": 0.91,
  "reason": "diagram chứa pump giống query"
}
```
